宽带连接错误代码720解决方案

solution
  • network

720：不能建立到远程计算机的连接。你可能需要更改此连接的网络设置。

客户端若出现这种错误，可以使用网络诊断推荐的方案：网络重置。五分钟后自动重启再试可能成功。
但若不成功，可以用第二种方法。

从设备管理器中找到网络适配器，卸载掉其中所有以WAN开头的驱动。
卸载完后再选择列表第一个选项（DESKTOP-85COU5J)，右击选择扫描检测硬件改动。再试应该就成功了！
